# epochcore-sdk

Public client SDK for the **EpochCore** platform — a thin wrapper around the EpochCore zero-latency single-API agent surface.

## Modules (planned)
- `dkap` — Decentralized Knowledge as Pixels client
- `epochos` — Orchestration & agent runtime hooks
- `epochpay` — Payment + billing-evidence client
- `epochcoverage` — Coverage / attestation helpers

## Status
Pre-release scaffold. Public surface only — all proprietary logic remains in private EpochCore infrastructure.
